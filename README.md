# 같이도전

같이 도전할만한 챌린지를 공유하고, 챌린지를 완수하면 챌린지에 참여한 구성원들에게 그
결과를 공유합니다.  
구성원이 관심있어 할 만할 챌린지가 만들어졌을 경우 알림으로 알려줍니다.   
또, 특정 결과를 내면 뱃지를 수여하여 참여자의 의욕을 올려줍니다.

https://github.com/Haru-piece
